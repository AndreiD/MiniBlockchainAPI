#### Send ETH & Tokens API

This small backend enables you to send ETH & Tokens to Ethereum based blockchains.

#### How to use it

GET    /api/v1/
GET    /api/v1/health
GET    /api/v1/balance/eth
GET    /api/v1/balance/token

PUT    /api/v1/tx/send_eth
PUT    /api/v1/tx/send_token
